# Barb-Bar LLC Sales Performance Analysis (Jan 2023 – July 2025)

## Project Overview
This project analyzes the sales performance of Barb-Bar LLC from January 2023 through July 2025. The purpose of this analysis is to support the company’s upcoming business relaunch after experiencing a significant decline in sales. By reviewing historical sales data, this project identifies what worked, what didn’t, and what strategies should be improved moving forward.

The analysis includes sales from both Shopify (online sales) and Square (in-person market sales).

 ## Business Objective
The main goal of this project is to:
- Understand overall sales performance
- Compare Shopify vs Square sales
- Identify revenue trends and channel effectiveness
- Evaluate Average Transaction Value (ATV)
- Support better business decisions for the Barb-Bar relaunch
  

## Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- XLOOKUP for KPIs
- Data Cleaning & Organization

- Dashboard Creation

## Data Sources
- Shopify Sales (Online)
- Square Sales (In-Person Markets)

Both datasets were cleaned, formatted, and merged into a single master spreadsheet to create unified reporting and dashboards.


## Key Analysis & Dashboards
- Total Sales by Year
- Sales by Channel (Shopify vs Square)
- Revenue Trends Over Time
- Average Transacion Value (ATV)
- Product & Market Performance
- KPI Tracking using XLOOKUP


## Final Deliverables
- Cleaned & Merged Excel Datasets
- Pivot Table Analysis
- Interactive Dashboards
- Final Business Presentation
- Written Report with Insights & Recommendations


## Key Business Insight
This analysis allows Barb-Bar LLC to clearly see which sales channels performed best, where revenue declined, and how customer purchasing behavior changed over time. These insights will be used to improve marketing strategy, product focus, and pricing during the company’s relaunch.
